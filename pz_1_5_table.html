МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис Стандарт
Основні теги (для каркасу документа) <html> Початок і кінець документа; вказує програмі перегляду сторінок що це HTML документ HTML5
<head> Заголовок; визначає місце, де поміщується різна інформація, яка не відображається в тілі документа. Тут розташовується тег назви документа та теги для пошукових машин HTML5
<body> Тіло документа; визначає видиму частину документа HTML5
Теги фізичного форматування тексту <p> Створює новий абзац HTML5
<pre> Тег авторського форматування. Виділяє попередньо відформатований і розмічений текст, тобто – рядки відображаються точно так само, як надруковані, і при цьому зберігаються множинні пробіли і переведення рядків HTML5
<b> Виділення жирним шрифтом (болд) HTML5
<i> Виділення нахиленим шрифтом (курсивом) HTML5
<u> Виділення підкресленням HTML5
<strike> Виділення переккреслюванням HTML5
<tt> Оформлення шрифтом з фіксованою шириною літер (моноширинним шрифтом); використовується рідко – приклад HTML5
<sub> Використовується для підрядкових індексів HTML5
<sup> Використовується для надрядкових індексів HTML5
<br> Вставлює примусове переведення на новий рядок HTML5
Теги логічного форматування тексту <h1> Створює найбільший заголовок HTML5
<h2> <h3> <h4> <h5> Створює заголовки проміжних розмірів HTML5
<h6> Створює найменший заголовок HTML5
<blockquote> Цитата; зазвичай відокремлюється пробілами зверху і знизу, а також відступом ліворуч, і відображається курсивом HTML5
<kbd> Текст, уведений користувачем HTML5
<font> Тег для роботи зі шрифтами тексту HTML5
Якірні теги та гіперпосилання <a href=”url”> Створює гіперпосилання на іншу веб-сторінку, інтернет ресурс або файл HTML5
<a href=”mailto:email”> Створює гіперпосилання виклику поштової програми для написання листа за вказаною адресою HTML5
<a href=”#мітка”> Створює гіперпосилання на мітку поточної сторінки HTML5
<a name=”мітка”> Зазначає частину тексту, як позначку для гіперпосилань на сторінці HTML5
<a href=”url#мітка”> Створює гіперпосилання на мітку іншої сторінки HTML5
Якірні теги та гіперпосилання <img src=”url”> Додає зображення в HTML документ HTML5
<hr> Додає в HTML документ горизонтальну лінію HTML5
<map name=”myname”> Задає зображення-карту, оремі частини якої є гіперпосиланням (чутливі області); поміщується в розділ head HTML5
<area> Задає чутливі області HTML5
Таблиці в HTML <table> Створює таблицю HTML5
<tr> Визначає рядок у таблиці HTML5
<td> Визначає окрему комірку в таблиці HTML5
<th> Визначає заголовок таблиці (нормальна комірка з відцентрованим жирним текстом) HTML5
<caption> Визначає підпис таблиці HTML5
